#### 🌑 [NYXIS](https://github.com/rajdangi31/NYXIS) — Gamified Behavioral OS
**"A logic-driven framework designed for structured personal evolution."**

<p align="center">
  <img src="https://raw.githubusercontent.com/rajdangi31/NYXIS/main/docs/assets/nyxis-banner.png" alt="NYXIS Banner" width="100%">

NYXIS is an experimental framework designed to function as an interactive agent. It monitors behavioral patterns, generates AI-driven directives, and reinforces consistency through system constraints.

* **Dual-Brain AI**: Implements a two-stage cognitive pipeline (Strategist → Generator) for context-aware task generation.
* **Behavioral Profiling**: Uses Exponential Moving Averages (EMA) to track Consistency, Avoidance, and Intensity metrics.
* **Systems Focus**: Real-time state persistence, memory-etching logic, and haptic-driven interactions.
* **Tech Stack**: Expo SDK 54, Supabase (Postgres/RLS/Edge Functions), OpenRouter (GPT-4o-mini).

#### 🧠 [GPT-2 from Scratch](https://github.com/rajdangi31/GPT-2) — Deep ML Fundamentals
**"Peeling back the abstraction layers of Modern Transformers."**

<p align="center">
  <img src="https://raw.githubusercontent.com/rajdangi31/GPT-2/main/loss_plot.png" alt="GPT-2 Loss Plot" width="600">
</p>

A ground-up implementation of the GPT-2 architecture using PyTorch primitives. This project serves as a technical showcase for understanding the raw mechanics that power large language models without using high-level APIs.

* **Custom Tokenizer**: Character-level manual ingestion of text into integer space.
* **Causal Self-Attention**: Implementing QKV matrices and lower-triangular causal masking (`tril`) for autoregressive learning.
* **Dialogue Generation**: Trained on classical dialogue datasets to learn semantically structured turn-taking.
* **Tech Stack**: Python, PyTorch, NumPy, AdamW Optimization.

#### 👀 [takeApeek](https://github.com/rajdangi31/takeApeek) — Cinematic Social Discovery
**"Real-time social discovery built with a Glass & Glow design system."**

<p align="center">
  <img src="https://raw.githubusercontent.com/rajdangi31/takeApeek/main/homepage_showcase.png" alt="takeApeek Showcase" width="100%">
</p>

A real-time social platform focused on high-fidelity interactions and cinematic motion. It balances minimalist aesthetics with complex serverless engineering.

* **Infinite Horizon**: Windowed fetching for sub-second infinite scroll performance.
* **Multi-Directional Flux**: Real-time synchronization using Supabase Channels and broadcast listeners.
* **Aesthetic Engineering**: Leveraging HSL tokens, spring-based physics (Framer Motion), and fluid sizing.
* **Tech Stack**: React 18, Vite 6, Tailwind CSS v4, TanStack Query v5, Supabase Cloud.
